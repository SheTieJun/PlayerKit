## PlayerKit

具备播放器基本功能，以及
1. 竖屏切换
2. 悬浮窗播放
3. 画质切换
4. 硬件加速
5. 倍速播放
6. 镜像播放
7. 手势控制
8. 支持直播与点播
9. 缓存播放
10. 截屏
11. 设置播放起始位置
12. 定时播放功能
13. 自动下一集功能
14. 自动开始播放
15. 界面控制
16. 播放回调
17. 雪碧图（暂时移除）、关键点
18. 自定义悬浮窗
